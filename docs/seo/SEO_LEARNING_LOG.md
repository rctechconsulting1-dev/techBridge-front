# SEO Learning Log

Purpose: Capture validated ranking and conversion lessons for built-in pages so the SEO agent and team can improve their output over time.

## How to use

Add short notes only when a pattern is validated by:

1. Real onboarding outcomes
2. Search performance feedback
3. Better conversion behavior
4. Repeated success across similar business types

Keep each note concrete and reusable.

## Entries

### 2026-04-02

1. For local service businesses, Home copy performs better when the hero body includes service category, geography, and a direct conversion path in the first paragraph.
2. Weak CTA text like Learn More should usually be replaced with call, estimate, booking, or service-request language when the business is lead-driven.
3. About pages should reduce trust friction by emphasizing credentials, process, and specificity instead of brand filler.
