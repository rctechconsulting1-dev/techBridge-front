# Phase 8 Tenant Launch Checklist

Date: 2026-03-24
Status: Active v1
Owner: RC Tech Bridge

Use this checklist for every new tenant before go-live.

## 1) Tenant Setup

- [ ] Tenant record created
- [ ] Tenant owner membership assigned
- [ ] Required modules enabled

## 2) Domain and Security

- [ ] Domain added and set primary
- [ ] DNS verification passed
- [ ] SSL and canonical routing verified

## 3) Email and Notifications

- [ ] Sender profile configured
- [ ] SPF/DKIM status verified
- [ ] Lead routing recipients validated

## 4) Payments and Commerce (if enabled)

- [ ] Stripe Connect onboarding completed
- [ ] Checkout test transaction passed
- [ ] Webhook delivery validated

## 5) Content and Branding

- [ ] Core pages published
- [ ] Branding tokens and logos applied
- [ ] Contact/booking intake tested

## 6) Reliability and Ops

- [ ] Reliability dashboard shows tenant activity
- [ ] Alert routing validated
- [ ] On-call escalation owner assigned

## 7) Handoff

- [ ] Client training completed
- [ ] Support runbook shared
- [ ] Go-live sign-off recorded
